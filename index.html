Linux Command Guide

Learn the essential Linux commands for system management and user control.

init useradd usermod userdel groupadd groupmod groupdel passwd sudo

init

Description: The init command is the first process that starts when a Linux system boots.

Example:

sudo init 3
×

useradd

Description: The useradd command is used to create a new user on the system.

Example:

sudo useradd -m -s /bin/bash john
×

usermod

Description: The usermod command modifies the properties of an existing user account.

Example:

sudo usermod -aG sudo john
×

userdel

Description: The userdel command is used to delete a user from the system.

Example:

sudo userdel -r john
×

groupadd

Description: The groupadd command is used to create a new group in the system.

Example:

sudo groupadd admins
×

groupmod

Description: The groupmod command modifies an existing group on the system.

Example:

sudo groupmod -n newadmins admins
×

groupdel

Description: The groupdel command is used to delete a group from the system.

Example:

sudo groupdel admins
×

passwd

Description: The passwd command is used to change the password of a user.

Example:

sudo passwd john
×

sudo

Description: The sudo command is used to execute commands with superuser (root) privileges.

Example:

sudo apt-get update
×
